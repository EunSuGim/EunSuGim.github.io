## HTML  
`HTML은 tag로 구성된 언어입니다.`  
tag에는 두가지 종류가 있습니다.  
- 시작태그와 닫는 태그로 구성된 태그  
	ex) `<title>Title</title>`  

- 시작태그만 가지고 있는 태그  
	ex) `<meta charset="UTF-8">`  
	
`tag는 중첩구조`를 가질 수 있습니다.  
하나의 tag가 다른 tag들을 포함할 수 있습니다.  
이때 부모자식, 후손관계가 성립됩니다.  
tag와 tag안에 포함된 요소들을 포함해서 element라고 부릅니다.
