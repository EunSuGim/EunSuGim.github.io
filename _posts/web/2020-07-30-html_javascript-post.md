## HTML  
HTML은 tag로 구성된 언어입니다.  
tag에는 두가지 종류가 있습니다.  
- 시작태그와 닫는 태그로 구성된 태그  
	ex) <title>Title</title>
- 시작태그만 가지고 있는 태그  
	ex) <meta charset="UTF-8">
